# User Guide Index

Folder ini berisi dokumentasi client untuk dua tipe business pada POSXPRESS.

## File Utama

- `UserGuide_Retail.md`: panduan pengguna untuk bisnis retail
- `UserGuide_Clinic.md`: panduan pengguna untuk bisnis clinic

## File Plan

- `PLAN_UserGuide_Retail.md`: struktur dan ruang lingkup dokumen retail
- `PLAN_UserGuide_Clinic.md`: struktur dan ruang lingkup dokumen clinic

## File Checklist Screenshot

- `SCREENSHOT_CHECKLIST_Retail.md`: daftar screenshot yang perlu disiapkan untuk retail
- `SCREENSHOT_CHECKLIST_Clinic.md`: daftar screenshot yang perlu disiapkan untuk clinic

## File Pendukung Tambahan

- `SCREENSHOT_NAMING_CONVENTION.md`: standar penamaan file screenshot
- `CHANGELOG_DOCUMENTATION.md`: catatan perubahan dokumentasi

## Saran Finalisasi Sebelum Dikirim ke Client

1. Ganti seluruh placeholder screenshot dengan screenshot aplikasi aktual.
2. Review istilah bisnis agar sesuai dengan istilah yang dipakai client.
3. Tambahkan logo company client pada halaman pembuka jika diperlukan.
4. Export markdown ke PDF jika client membutuhkan versi cetak.
5. Lakukan review akhir bersama tim implementasi atau PIC produk.
