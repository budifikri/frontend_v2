## Brief overview
Pedoman penggunaan MCP Playwright untuk otomasi browser dan testing. File ini berisi konfigurasi dan fitur-fitur yang tersedia untuk MCP Playwright.

## Konfigurasi MCP
- MCP Playwright dikonfigurasi secara global
- Menggunakan Node.js dengan package playwright
- Tersedia fitur screenshot dan form filling

## Fitur yang tersedia
- **screenshot**: Mengambil screenshot halaman web
- **form_filling**: Mengisi form otomatis
- **navigate**: Navigasi ke URL
- **click**: Klik elemen pada halaman
- **type**: Mengetik teks ke input field
- **wait**: Menunggu elemen muncul
- **get_text**: Mengambil teks dari elemen

## Penggunaan
- Gunakan fitur ini untuk testing otomasi browser
- Cocok untuk testing form dan validasi UI
- Dapat digunakan di semua project
