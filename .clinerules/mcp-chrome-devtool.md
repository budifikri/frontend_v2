## Brief overview
Pedoman penggunaan MCP Chrome DevTools untuk debugging dan inspeksi browser. File ini berisi konfigurasi dan fitur-fitur yang tersedia untuk MCP Chrome DevTools.

## Konfigurasi MCP
- MCP Chrome DevTools dikonfigurasi secara global
- Menggunakan Node.js dengan package chrome-devtools
- Tersedia fitur debugging dan inspeksi element

## Fitur yang tersedia
- **inspect_element**: Inspeksi element HTML
- **debug_js**: Debug JavaScript code
- **network_monitor**: Monitor network request
- **console_log**: Akses console log
- **performance**: Analisis performance
- **memory**: Analisis memory usage

## Penggunaan
- Gunakan fitur ini untuk debugging aplikasi web
- Cocok untuk inspeksi element dan analisis performance
- Dapat digunakan di semua project web
